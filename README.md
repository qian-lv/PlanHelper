# PlanHelper — 个人计划管理工具

一个纯前端单页应用，融合长期目标管理、每日计划执行与进度可视化，通过可交互的路线图画布帮助个人构建学习路径。

## 功能

- **长期计划** — 目标 → 阶段 → 任务三层结构，进度条可视化
- **今日计划** — 8:00-23:00 时间线（30 分钟一格），拖拽安排任务，打勾完成
- **进度路线图** — 可缩放/平移的大画布，已完成任务以卡片自由摆放，手动拉线连接形成学习路线
- **日历** — 月历视图，显示每日完成率
- **日常惯例** — 自动生成每日重复任务（跑步、背单词等），无需手动创建
- **数据管理** — 导入/导出 JSON，数据清空

## 使用方式

用浏览器直接打开 `index.html` 即可，无需安装任何依赖，无需本地服务器。数据存储在浏览器 localStorage 中。

## 技术栈

纯 HTML + CSS + JavaScript，零外部依赖，单文件约 1500 行。

- CSS3 Transform 实现画布缩放（0.3x-3x）与平移
- SVG 动态绘制任务连线
- HTML5 Drag & Drop 实现拖拽交互
- Store 模式管理数据读写与持久化
- 内置数据版本迁移（v1 → v2 → v3）
